One Side Slider

Create a Slider:

Use classes one-side-slider, track and slider to apply sliders style.

HTML

<div class="one-side-slider" id="my-slider">
    <div class="track"></div>
    <input type="range" class="slider"/>
</div>

Customize Colors

CSS

.one-side-slider#my-slider{
    --track-fill-color:  rgb(25, 227, 82);
    --track-nofill-color:  rgb(160, 255, 169);
    --thumb-color:  rgb(255, 255, 255);
    --thumb-border-color:  rgb(25, 227, 82);
    --active-thumb-color:  rgb(25, 227, 82);
    --active-thumb-border-color:  rgb(255, 255, 255);
}

Customize Shape

CSS

.one-side-slider#my-slider{
    --thumb-border-color:  rgb(197, 63, 99);
    --track-border-radius: 1rem;
    --thumb-width: 2rem;
    --thumb-height: 2rem;
    --track-height: 2rem;
}
