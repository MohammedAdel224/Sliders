Sliders

Get Strated With Sliders

Step 1: Download Sliders Files

  • Download the CSS file ( sliders.css or sliders.min.css ).
  • Download the JavaScript file ( sliders.bundle.js or sliders.bundle.min.js ).

Download from here:

https://github.com/MohammedAdel224/Sliders/tree/gh-pages

Step 2: Include Sliders To Your Project

  • Include CSS:
    <link rel="stylesheet" href="file-path/sliders.min.css">
  • Include JavaScript:
    <script src="file-path/sliders.bundle.min.js">

One Side Slider

Create a Slider:

Use classes one-side-slider, track and slider to apply sliders style.

HTML

<div class="one-side-slider" id="my-slider">
    <div class="track"></div>
    <input type="range" class="slider"/>
</div>

Customize Colors

CSS

.one-side-slider#my-slider{
    --track-fill-color:  #0013a9;
    --track-nofill-color:  #000000;
    --thumb-color:  #ffffff;
    --thumb-border-color:  #0013a9;
    --active-thumb-color:  #0013a9;
    --active-thumb-border-color:  #ffffff;
}

Customize Shape

CSS

.one-side-slider#my-slider{
    --track-height: 2rem;
    --track-border-radius: 1rem;
    --thumb-width: 2rem;
    --thumb-height: 2rem;
    --active-thumb-color:  #630155;
    --thumb-border-width: 0px;
}

Default Style

CSS

.one-side-slider{
    --fill-percent: 50%; /*this value will be calculated using JavaScript*/
    --slider-pointer-events: none;

    --track-border-radius: 5px;
